Active Session 레이아웃 UX 개선

분석 결과 겹침 해소 + 리포트 편집 UX + 플로우 시각화

문제

  • ✖ Draft Report가 로그를 완전히 가림 — absolute overlay로 겹침, 두 정보를 동시에 볼 수 없음
  • ✖ 화면 피드 70% / 사이드바 30% — 분석 후에는 피드보다 결과가 중요한데 공간 부족
  • ✖ 리포트 편집 불편 — textarea 3줄, 좁은 width에서 내용 확인/수정 어려움
  • ✖ 분석 진행 상태 불명확 — 캡처 → 분석 → 결과 단계가 시각적으로 구분 안 됨
